## **随手记**

Caddy 是一个多功能的 HTTP web服务器，并且使用Let's Encrypt提供的免费证书，自动让网站升级到HTTPS
为什么要使用 Caddy      
安全： Caddy 是一个默认使用https协议的web服务器        
无依赖：Caddy 使用 Go 语言编写，编译好的二进制文件能够运行在任何支持Go语言的平台，不需要自己安装任何库。        
使用简单：Caddy 的配置简单，不管你是新的web开发者，还是专业人士，都能够快速上手
